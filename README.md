CountryGamer_Oceanic
====================
Oceanic Mod

This is an Open Source Minecraft mod/plugin, made by TheCountryGamer
The OceanicMod added the following:
Items:
- Saturated Leather
- Water Resistant Leather
- Aerate Piece
- Aerate Ingot

Armor:
- Watersuit Armor
- Diving Suit Armor

Blocks:
- Aerate Ore

General Achievements (Hidden)
